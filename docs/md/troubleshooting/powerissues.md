****** Power-Related Issues ​ ******
If your Electra One controller shows unusual behavior during startup or
operation, it may be caused by insufficient power supply. This can happen when
using a low-quality USB cable, a long USB cable, or a MIDI USB host (e.g.
router) that does not provide enough current.
***** Symptoms ​ *****
    * Controller repeatedly restarts - The device continuously reboots shortly
      after completing the startup animation.
    * White screen - The screen turns completely white and does not recover.
    * Phantom touches or display glitches - Random or erratic input appears on
      the screen, including flickering or ghost touches.
***** Recovery Procedure ​ *****
   1. Use a better-quality USB cable
          o Replace the current USB cable with a shorter, high-quality cable
            capable of delivering stable power. Avoid excessively long cables
            or thin, unshielded ones, as they may cause voltage drops.
   2. Improve power delivery on the USB Host port
      If you're using a MIDI router or USB host that does not supply enough
      current:
          o Use a powered USB hub between the USB host and Electra One to
            supply additional power.
          o If your USB host does not support USB hubs, consider using a USB Y-
            cable to split power and data. This allows you to power the Electra
            One from a separate USB power adapter while maintaining the data
            connection.
   3. Reduce display brightness
      Lowering the LCD backlight level can reduce overall power consumption.
          o On the Electra One controller, go to: Menu → Settings → Interface
            tab
          o Reduce the brightness level to minimize power draw.
***** Further Assistance ​ *****
These steps should resolve most issues related to power instability. If you
continue experiencing display problems or restarts, please post in the Electra
One_Community_forum or contact Electra One support via email.
